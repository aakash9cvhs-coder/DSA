A **Java console-based Event Management System** built using **Data Structures and Algorithms (DSA)** concepts.  
The project simulates how a university manages events, registrations, waiting lists, and event priorities.

---

# 📌 Project Overview

UniEvents allows students to:

✔ View available university events  
✔ Search events by ID  
✔ Register for events  
✔ Manage waiting lists when events are full  
✔ Track registration history  
✔ View priority events  

This project demonstrates **practical implementation of multiple DSA concepts in one system.**

---

# 🧠 Data Structures Used

| Data Structure | Usage |
|---------------|------|
| Doubly Linked List | Store and manage events |
| Stack | Registration history |
| Circular Queue | Waiting list for full events |
| Hash Table | Store and search users |
| Priority Queue | Event priority management |
| Stack Algorithm | Infix to Postfix conversion |

---

# ⚙️ Features

### 📋 View Events
Displays all available events with **ID, name, and remaining seats**.

### 🔍 Search Event
Search events quickly using **Linear Search**.

### 📝 Register Event
Students can register if seats are available.

### ⏳ Waiting Queue
If seats are full, students are automatically added to the **Circular Queue waiting list**.

### 📜 Registration History
Tracks previously registered events using a **Stack**.

### ⭐ Priority Events
Shows events based on priority using a **Priority Queue**.

### 🧮 Expression Conversion
Implements **Infix to Postfix algorithm using Stack**.
